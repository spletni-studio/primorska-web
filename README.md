# Primorska Web

Enostavna predstavitvena spletna stran za ponudbo izdelave spletnih strani, prenove obstoječih strani in manjših spletnih rešitev.

Projekt je namenjen predstavitvi storitev za lokalna podjetja v Sloveniji, predvsem na Obali in v Primorski regiji.

## Namen

Stran je pripravljena kot osnovna predstavitev storitev:

- izdelava spletnih strani
- prenova obstoječih strani
- manjše spletne trgovine
- popravki in optimizacija
- vzdrževanje spletnih strani

Cilj projekta je preprosta in pregledna predstavitev za manjša podjetja, obrtnike, gostince in storitvene dejavnosti.

## Tehnologija

Projekt uporablja:

- HTML
- CSS
- osnovni JavaScript

Gre za statično spletno stran brez dodatnih ogrodij.

## Deploy

Stran je namenjena objavi na statičnem hostingu, kot so:

- Netlify
- Vercel
- Cloudflare Pages
